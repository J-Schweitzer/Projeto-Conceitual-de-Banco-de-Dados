E-Commerce Database

Este projeto é um banco de dados relacional desenvolvido para um sistema de e-commerce. O objetivo é fornecer uma estrutura que gerencie eficientemente informações sobre clientes, produtos, pedidos, fornecedores, estoque, transportadoras e pagamentos. Este esquema de banco de dados foi projetado para suportar as operações principais de um sistema de e-commerce, garantindo consistência nos dados e possibilitando o armazenamento de várias formas de pagamento, acompanhamento de entregas e diferenciação entre clientes pessoa física e jurídica.


Objetivo:
Refine o modelo apresentado acrescentando os seguintes pontos:

Cliente PJ e PF – Uma conta pode ser PJ ou PF, mas não pode ter as duas informações;
Pagamento – Pode ter cadastrado mais de uma forma de pagamento;
Entrega – Possui status e código de rastreio;
